# Cryptand

> **让 Minecraft 整合包跑得更流畅**

**Cryptand** 是一个 Minecraft **联动优化型模组**，围绕两个核心目标：

1. **模组联动** — 在不同的大型模组之间搭建桥梁，拓展更多玩法
2. **性能优化** — 帮助这些模组跑得更快、更稳定，减少卡顿

它本身不添加新物品或新玩法，专注于幕后工作。

---

## ✅ 目前已实现的优化

### ⚡ PowerGrid 电网仿真 — 多线程加速

当你的整合包安装了 [PowerGrid](https://www.curseforge.com/minecraft/mc-mods/power-grid)（电力网络模组）时，电网仿真计算会消耗大量 CPU，导致游戏卡顿。

PowerGrid 自带硬件加速能力，但在多线程环境下容易崩溃。Cryptand 所做的：

- **多核并行**：将电网计算拆分到多个 CPU 核心同时运行，不再卡主线程
- **稳定性修复**：修复了多线程下硬件加速会崩溃的问题，让原有加速能力安全生效
- **安全保护**：设有超时机制，即使计算卡死也不会拖垮整个服务器
